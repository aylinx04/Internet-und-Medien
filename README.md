# Internet und Medien

## Übersicht

In diesem Modul, wurde in Einzelarbeit ein Webseite realisiert mit HTML, CSS, JavaScript und Node.js. In diesem Projekt habe ich mich für eine Webseite entschieden, in welchem Videospiele bewertet werden. 

## Architektur

```bash
\---Projekt
    |   .project
    |   package.json
    |   server.js
    \---public
            Bestlist.html
            Forum.html
            Login.html
            project.js
            Reviews.html
            StartSeite.html
```
In der Architektur sieht man mehrere HTML Seiten, CSS ist bereits in den HTML integriert und eine JavaScript für das dynamische Design. Mit Node.js Server lastiges, simple Anmeldung implementiert.


## Starten der Webseite

Hierfür muss im Terminal ```node server.js``` eigegeben werden. Folgendes sollte angezeigt werden:

```bash
Server läuft auf http://localhost:8000
```
